# AI 解題檢查單｜講義草稿 v1

用途：長片附件、課堂活動單、下一支影片的核心素材。建議一頁 A4，可讓學生拿一份 AI 生成的物理解答來做批判性檢查。

## 使用方式
1. 把題目原文貼在左欄。
2. 把 AI 解答貼在右欄。
3. 依序檢查四件事：條件、方向、單位、意義。
4. 最後請學生寫下「我會怎麼修正這份解答」。

---

## ① 條件檢查：題目給了什麼？AI 用了什麼？

| 檢查問題 | 勾選 | 證據／註記 |
|---|---:|---|
| AI 有列出題目給定量嗎？ | ☐ |  |
| AI 有新增題目沒有給的假設嗎？ | ☐ |  |
| AI 有忽略題目明講的限制嗎？ | ☐ |  |
| AI 使用的模型和題目情境一致嗎？ | ☐ |  |

常見錯誤：把「等速」當成「等加速度」、把「忽略空氣阻力」又偷偷加入阻力、把「由靜止釋放」改成有初速度。

## ② 方向檢查：正負號代表什麼？

| 檢查問題 | 勾選 | 證據／註記 |
|---|---:|---|
| AI 有明確定義正方向嗎？ | ☐ |  |
| 力、速度、加速度方向有一致嗎？ | ☐ |  |
| 公式中的正負號和圖像方向一致嗎？ | ☐ |  |
| 最後答案的正負號有被解釋嗎？ | ☐ |  |

老師追問句：`如果把正方向改掉，答案的意義會怎麼變？`

## ③ 單位與量級檢查：答案像不像真實世界？

| 檢查問題 | 勾選 | 證據／註記 |
|---|---:|---|
| 最後答案單位正確嗎？ | ☐ |  |
| 中間式子的左右單位一致嗎？ | ☐ |  |
| 數值大小合理嗎？ | ☐ |  |
| 是否出現日常情境不可能的結果？ | ☐ |  |

快速判斷：力 N、能量 J、功率 W、速度 m/s；單位不對時，通常不用重算就知道有問題。

## 教師延伸活動
- 小組 A 專查條件，小組 B 專查方向，小組 C 專查單位，小組 D 專查物理意義。
- 讓學生先投票 AI 哪裡最可能錯，再公布老師版本。
- 把錯誤答案當成「診斷題」，訓練學生說明為什麼錯。
