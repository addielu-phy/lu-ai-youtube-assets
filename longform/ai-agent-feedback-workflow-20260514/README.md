## 5 張 storyboard
1. Hook：AI 不是代寫評語，而是進入審稿線。
2. 輸入邊界：先餵匿名資料，不餵學生身分。
3. AI 草稿包：全班共通回饋、分組補救任務、教師需確認風險句。
4. 教師審稿線：三色筆審稿，刪個資／改證據／補脈絡。
5. 45 分鐘課後回饋流程：匿名摘要、AI 草稿、教師定稿、下節課再用。
