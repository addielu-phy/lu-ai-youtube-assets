定位：第一季優先題 2；承接 Shorts 01「AI 算錯？老師抓包！」與長片檢查點「單位與量級」。  
目標：用 30–35 秒讓老師與學生記住：AI 答案看起來完整時，不要先被步驟說服；先看最後單位、中間式子與真實量級。

## 1. 核心一句話

AI 解物理題時，單位一對不上，不一定要重算整題；先按暫停，因為它可能已經把物理量搞混了。

## 2. 建議片長與節奏

- 目標片長：30–35 秒
- 規格：9:16、1080×1920、30fps
- 風格：深藍科技底、黃色「先暫停」提示、紅色錯誤單位警示
- 人設：老師用最快的物理直覺把 AI 解答先篩過，不是反 AI，而是教學生不要盲抄

```text
最後單位對不對？

答案量級像不像真實世界？

AI 可以當助教，但單位不過關，這份解答就不能直接交給學生。
```
