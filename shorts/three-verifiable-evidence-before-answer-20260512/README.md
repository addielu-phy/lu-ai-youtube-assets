# Shorts storyboard｜AI 給答案前，先叫它列出三個可驗證證據

狀態：2026-05-12 每小時雷達自動推進完成 storyboard package v1。  
來源：第三季內容題庫優先題 1。

## 30 秒製作簡報

- 3 秒鉤子：答案先放旁邊，證據在哪？
- 核心觀點：AI 回答前先列三個可驗證證據，學生才有基準判斷答案是不是只是在「講得很順」。
- 對象：國高中自然／物理教師，可用於 5 分鐘暖身、AI 解題審稿或證據評分規準延伸活動。
- 最小素材：5 張 9:16 圖卡、三欄證據卡、老師追問句。
